# คำอธิบายโซลูชันสำหรับการหาคำในกระดาน (Word Search)

## ภาพรวมของโซลูชัน

โซลูชันนี้ใช้สำหรับการหาคำในกระดาน 2 มิติ โดยคำสามารถอยู่ได้ในทิศทาง 8 ทิศทาง (แนวนอน แนวตั้ง และแนวทแยง) โดยใช้การค้นหาแบบ Iterative Depth-First Search (DFS) เพื่อหลีกเลี่ยงการเรียกซ้ำ (Recursion) ซึ่งอาจทำให้เกิด Stack Overflow ในกรณีที่คำยาวหรือกระดานใหญ่

### โครงสร้างหลัก

- **คลาส FinalSolution**: ใช้สำหรับการหาคำในกระดาน
- **เมธอด EXAM01_FindWords**: เมธอดหลักที่รับกระดานและรายการคำ แล้วหาคำที่พบในกระดาน
- **เมธอด FindWord**: เมธอดช่วยในการค้นหาคำหนึ่งคำโดยใช้ Iterative DFS
- **คลาส PathNode**: คลาสสำหรับเก็บสถานะของการค้นหาในแต่ละขั้นตอน
